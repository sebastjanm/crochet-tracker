# Artful Space Crochet Tracker

Cross-platform mobile app for crochet enthusiasts to manage projects, inventory, and access AI features.

---

## Key Features

### Projects
- Track projects by status: Idea, In Progress, Completed, On Hold, Archived, Frogged
- Multiple images, pattern PDFs, inspiration URLs
- Notes, yarn usage tracking

### Inventory
- **Yarn**: brand, weight, composition, color, gauge
- **Hooks**: sizes, materials, ergonomic features
- **Other**: stitch markers, scissors, needles, etc.
- Quantity tracking, location tags, project linking

### YarnAI
- Chat assistant for crochet questions
- Project idea generator
- Image generator for inspiration
- Voice assistant

### Profile
- Authentication (Supabase)
- Multi-language support (EN, DE, SL, RU)
- Customizable avatars
